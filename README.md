# Credit Card Approval Prediction

## Project Overview
This project is a Machine Learning web application that predicts whether a credit card application will be approved or rejected based on user input data.

## Tech Stack
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- HTML, CSS

## Model Used
Random Forest Classifier (Best Performing Model)

## Dataset
Dataset stored in Google Drive (due to large size):
https://drive.google.com/file/d/13Yr0asj8gXFI1oQTFWWo557wJlna9s2x/view?usp=drive_link

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run application:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
