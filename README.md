# Credit Card Approval Prediction Using Machine Learning

## 📌 Project Overview

The Credit Card Approval Prediction project is a Machine Learning-based web application that predicts whether a credit card application should be approved or rejected based on applicant information. The system helps financial institutions automate the approval process, reduce manual effort, and improve decision-making.

---

## 🎯 Objectives

- Automate the credit card approval process.
- Improve prediction accuracy.
- Reduce manual verification time.
- Minimize human errors.
- Provide a user-friendly web interface.

---

## 🚀 Features

- Credit card approval prediction
- User-friendly Flask web application
- Machine Learning-based decision making
- Fast and accurate predictions
- Simple interface

## ▶️ How to Run

1. Clone the repository

```bash
git clone <repository-link>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```

4. Open the browser

```
http://127.0.0.1:5000
```
